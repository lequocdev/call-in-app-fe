# Call-In-App Frontend

This workspace contains the frontend implementation for the Twilio App-to-App Voice Call Simulator.

## Repository Structure
- `apps/voice-call-simulator/` - Single-Device Twilio Voice call simulator UI.
  - Built with: **Vite + TypeScript + Vanilla CSS + Twilio Voice SDK (NPM)**.
  - Features: Manual credentials form, dynamic role selector, visual idempotency key generation, detailed console logs.
- `voice_simulator_fe_plan.md` - Complete frontend specification plan.
